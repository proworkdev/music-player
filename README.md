# firebase

#### dependencies

```
npm i firebase

npm i react-firebaseui
```

---------------------------------------------------------------------------------------------------------------------------------------

#### How to start project

```
npm install
```

```
npm start
```

or

```
HTTPS=true npm start
```

URL: https://localhost:3000/
